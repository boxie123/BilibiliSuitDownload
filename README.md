## BilibiliSuitDownload

下载装扮中所有的https资源(大部分为图片)

> 最新版本已支持爬取收藏集

### 使用方法

1. 运行 `release` 中的 `.exe` 文件
1. 输入要爬取的装扮的分享链接
   > ~短链接的跳转通过js脚本进行，若印入playwright则程序过于笨重，所以暂时不支持。~
   >
   > ~仅支持`hostname`为`www.bilibili.com`的链接~
   > 
   > 目前已支持全部分享链接（包括短链接）
1. 回车运行
