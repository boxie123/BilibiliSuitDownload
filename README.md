## BilibiliSuitDownload

下载装扮/收藏集中所有的https资源(大部分为图片/视频)

### 使用方法

下载并运行 [`release`](https://github.com/boxie123/BilibiliSuitDownload/releases/latest) 中的 `.exe` 文件，根据文字提示操作
